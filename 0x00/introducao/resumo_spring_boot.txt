Spring Boot Framework é um framewrok Java para desenvolvimento de microsserviços. Os benefícios de utilizá-lo é não ter de fazer configurações do projeto do zero, devido ao fato do framework adotar um modelo “convenção sobre configuração”, auxílio com as dependências do projeto, além de contar com servidor embarcado, por padrão, o Apache Tomcat. O Spring Boot possui uma dependência responsável por gerir subdependências, ou seja, ao adicionar alguma dependência no projeto, automaticamente são adicionadas as outras dependências para que o projeto funcione. O Spring Tools pode ser utilizado em qualquer IDE, mas a comunidade Spring Tools Suite para trabalhar com Spring Boot. Spring Initializr é um gerador de projetos baseados em JVM com diversas implementações, como por exemplo os projetos Maven.
